# Side project 3

## doitt white

> backend

```bash
yarn create strapi backend 
```

> 啟動backend

```bash
cd backend
yarn run develop
````

> frontend
> 使用 vite 建置

```bash
yarn create vite
```

> 直接命名專案名稱: frontend

```bash
yarn create vite frontend --template vue
```

> 啟動frontend

```bash
cd frontend
yarn install
yarn dev
```
